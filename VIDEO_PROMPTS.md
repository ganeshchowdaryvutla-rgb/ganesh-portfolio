# 🎬 Video Generation Prompts for Ganesh Portfolio

Use these prompts with **Google Whisk + Veo 3**, **Runway Gen-4**, or **Pika** to generate
cinematic video content for your portfolio website.

---

## Silent Portfolio Hero Video (Recommended)

> Best for: Hero section background loop

```
Create a 20-30 second seamless looping cinematic video. A young Indian
entrepreneur in a white formal shirt and navy blue trousers stands in a
dark luxury environment with a soft overhead spotlight. No voice, no text,
no subtitles.

Scene flow:
1. Character stands confidently, subtle breathing animation, camera slowly
   pushes in, floating orange particles in background
2. Character walks toward camera with natural body movement, smooth
   cinematic tracking shot
3. Character stops near camera, looks directly forward, slight smile,
   confident posture, small natural hand gestures
4. Futuristic holographic panels appear around character, transparent glass
   interfaces, floating charts, digital particles, AI-inspired visuals
5. Camera circles around character, soft rim lighting, depth of field,
   premium reflections, orange ambient glow
6. Character walks forward again, camera pulls back, environment expands,
   seamless loop back to start

Style: Apple Vision Pro commercial, Tesla launch event, OpenAI reveal.
4K, 60fps, ultra realistic, cinematic lighting, depth of field, dark
background, no audio, no captions, seamless loop.
```

## Cinematic Intro Video (With Voice - for Social Media)

> Best for: LinkedIn, Instagram, YouTube shorts

```
Create a 30-45 second ultra-premium cinematic founder introduction video.

Scene 1 (0-8s): Dark futuristic environment, young Indian entrepreneur in
white shirt slowly walks toward camera, soft spotlight, floating orange
particles, cinematic dolly, text appears "Hi, I'm Ganesh"

Scene 2 (8-15s): Character stops, looks at viewer, hand gestures, holographic
interface appears with floating keywords: AI, Innovation, Startups, Technology

Scene 3 (15-22s): Character points toward floating holographic screens showing
premium website interfaces, 3D dashboards, code streams, UI designs

Scene 4 (22-30s): Environment transforms into AI laboratory, neural networks,
glowing data streams, digital particles orbiting character

Scene 5 (30-38s): Glass project cards appear, camera rotates around character

Scene 6 (38-45s): Character walks forward, futuristic city background, orange
skyline, camera rises, text "Building The Future"

Style: Apple Vision Pro + Tesla + OpenAI, 4K, 60fps, cinematic lighting.
```

## Negative Prompts (Add to All)

```
Avoid: cartoon style, low quality, fast movements, camera shaking,
unrealistic body motion, facial distortion, identity changes between shots,
bright backgrounds, casual clothing
```

---

## Tips

1. **Upload your avatar frames** (especially frames 001, 050, 100) as reference images
2. **Use the same prompt** across services for consistency
3. **For seamless loops**, trim the last 0.5s in a video editor and crossfade
4. **Vertical (9:16)** for mobile hero section, **Horizontal (16:9)** for desktop
5. **Export as WebM** for best web performance with transparency support
